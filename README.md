Sistema de Gestão: Sociedade Esportiva PalmeirasEste projeto é uma aplicação Web Fullstack desenvolvida para centralizar informações sobre a história, elenco e curiosidades da Sociedade Esportiva Palmeiras. O sistema permite que o torcedor interaja com dados do clube, participe de um quiz temático e acompanhe o desempenho através de uma dashboard personalizada.💚 Sobre o ProjetoEste sistema é mais do que uma aplicação de gestão; é uma homenagem à história do meu time de coração. O projeto nasceu da vontade de unir minha trajetória acadêmica na São Paulo Tech School com a paixão pelo alviverde, influenciada desde cedo pelo meu pai.O sistema permite cadastro e login de usuários, garantindo uma experiência personalizada para cada torcedor.🛠 Tecnologias UtilizadasPara garantir uma estrutura robusta e performática, utilizei:Frontend: HTML5, CSS3, JavaScript.Backend: Node.js.Banco de Dados: MySQL.Infraestrutura: Uso de Máquinas Virtuais (Virtualização) para o deploy e execução do ambiente.🚀 Como Executar o ProjetoClone este repositório:Bashgit clone [link-do-seu-repositorio]
Configuração do Banco de Dados:Execute os scripts disponíveis em /src/database/script-tabelas.sql.Parametrização:No arquivo app.js, ajuste o ambiente_processo para 'desenvolvimento' ou 'producao' conforme a sua necessidade.Configure as credenciais do seu banco de dados no arquivo .env ou .env.dev.Instalação e Execução:Bashnpm i
npm start
